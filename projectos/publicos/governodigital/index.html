Cinco superfícies de produto, um Estado digital.

Governo Digital Moçambique reúne identidade civil, requerimentos sem balcão, atendimento ao cidadão, transparência orçamental e auditoria de acessos numa única plataforma cívica. Estes mocks demonstram fluxos ponta-a-ponta para revisão de stakeholders antes da implementação.

Como navegar este pacote

1. Sistema de Design

Comece pelo Sistema de Design — define paleta, tipografia, componentes e estados que todas as superfícies reutilizam.

2. Superfícies clicáveis

Cada superfície abre num documento próprio com navegação interna. Botões avançam para o próximo estado; ramos divergentes (rejeição, escalonamento, override) estão acessíveis.

3. Documentos DESIGN-*.md

Cada superfície tem um documento autoritativo (ex.: DESIGN-ebi.md) descrevendo princípios, fluxos, componentes, estados e abertos. Para handoff a engenharia.

Governo DigitalPlataforma Cívica Nacional

Protótipo de demonstração criado por Hélder Monteiro para imaginar um sistema end-to-end de governo digital para Moçambique. A "Unidade de Governação Digital" é uma instituição fictícia neste exercício. Este protótipo não usa dados reais. Nomes, NUITs e BIs são ilustrativos.
