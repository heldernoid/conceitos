# Educação Digital · Conceitos

Reimaginação da plataforma nacional de educação com cinco superfícies: gestão escolar integrada, portal do professor, caderno electrónico do aluno, biblioteca pública nacional e portal de matrículas.

[→ Abrir protótipo](https://conceitos.org/projectos/publicos/educacaodigital/)

## Estrutura

```
educacaodigital/
├── index.html           # Visão geral e navegação
├── sige.html            # SIGE — Gestão Escolar
├── professor.html       # Portal do Professor
├── aluno.html           # Caderno do Aluno
├── biblioteca.html      # Biblioteca Nacional
├── matriculas.html      # Portal de Matrículas
├── design-system.html   # Sistema de Design
├── shared/styles.css
└── DESIGN-*.md          # Documentação de design por superfície
```
