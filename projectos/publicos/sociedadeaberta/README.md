# Sociedade Aberta · Conceitos

Reimaginação do acesso à informação pública com cinco superfícies: portal do cidadão, portal da empresa, ferramentas para jornalistas, plataforma de dados abertos e portal do titular de dados.

[→ Abrir protótipo](https://conceitos.org/projectos/publicos/sociedadeaberta/)

## Estrutura

```
sociedadeaberta/
├── index.html           # Visão geral e navegação
├── cidadao.html         # Portal do Cidadão
├── empresa.html         # Portal da Empresa
├── jornalista.html      # Portal do Jornalista
├── publica.html         # Plataforma Pública
├── titular.html         # Portal do Titular
├── design-system.html   # Sistema de Design
├── shared/styles.css
└── DESIGN-*.md          # Documentação de design por superfície
```
