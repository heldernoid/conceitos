Como navegar este pacote

1. Sistema de Design

Comece pelo Sistema de Design — define paleta, tipografia, componentes (NUIT card, factura, QR) que todas as superfícies reutilizam.

2. Superfícies clicáveis

Cada superfície abre num documento próprio com navegação interna. Botões avançam para o próximo estado; ramos divergentes (rejeição, fiscalização, recurso) estão acessíveis.

3. Documentos DESIGN-*.md

Cada superfície tem um documento autoritativo (ex.: DESIGN-efactura.md) descrevendo princípios, fluxos, componentes, estados e abertos. Para handoff a engenharia.

Finanças DigitaisPlataforma Tributária Nacional

Protótipo de demonstração criado por Hélder Monteiro para imaginar um sistema end-to-end de tributação digital para Moçambique. O Ministério da Economia e Finanças (MEF) e a Autoridade Tributária (AT) são entidades reais; a "Direcção de Finanças Digitais" (DFD) é uma unidade operativa fictícia neste exercício. Este protótipo não usa dados reais. Nomes de contribuintes, NUITs, valores e empresas são ilustrativos.
