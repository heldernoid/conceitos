Quatro superfícies de produto, um sistema de saúde.

Saúde Moçambique reúne registo clínico, atendimento ao cidadão, dispensa farmacêutica e auditoria de acessos numa única plataforma pública. Estes mocks demonstram fluxos ponta-a-ponta para revisão de stakeholders antes da implementação.

Como navegar este pacote

1. Sistema de Design

Comece pelo Sistema de Design — define paleta, tipografia, componentes e estados que todas as superfícies reutilizam.

2. Superfícies clicáveis

Cada superfície abre num documento próprio com navegação interna. Botões avançam para o próximo estado; ramos divergentes (dispensa parcial, escalonamento) estão acessíveis.

3. Documentos DESIGN-*.md

Cada superfície tem um documento autoritativo (ex.: DESIGN-emr.md) descrevendo princípios, fluxos, componentes, estados e abertos. Para handoff a engenharia.
