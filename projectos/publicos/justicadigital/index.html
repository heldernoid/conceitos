Como navegar este pacote

1. Sistema de Design

Comece pelo Sistema de Design — define paleta slate, tipografia, componentes (Process card, Acta serifada, QR) que todas as superfícies reutilizam.

2. Superfícies clicáveis

Cada superfície abre num documento próprio com cenas numeradas. Ramos divergentes (mediação falha, conflito sobreposição) acessíveis por cross-link.

3. Documentos DESIGN-*.md

Cada superfície tem um documento autoritativo (ex.: DESIGN-comunitario.md) com princípios, fluxos, componentes, estados. Para handoff a engenharia.

Justiça DigitalPlataforma Judicial Nacional

Protótipo de demonstração criado por Hélder Monteiro para imaginar um sistema end-to-end de justiça digital para Moçambique. O Ministério da Justiça, Assuntos Constitucionais e Religiosos (MJCR), o Tribunal Supremo, a PGR, a OAM e a DNRN são entidades reais; a "Direcção de Justiça Digital" (DJD) é uma unidade operativa fictícia neste exercício. Os tribunais comunitários estão previstos na Lei 4/92. Este protótipo não usa dados reais — nomes, NUITs, números de processo e factos são ilustrativos.
